Documentation Utilisateur - QuadraFlow

Présentation

QuadraFlow est une application de bureau permettant de traiter automatiquement les e-mails reçus dans un dossier spécifique, d'extraire les pièces jointes, de les classer dans les bons dossiers du Drive, et de mettre à jour les tâches dans Asana.

Fonctionnalités

  • Lecture des e-mails depuis la boîte immatriculation@carington.fr, dossier 02_SIGNATURE - ASSURANCE
  • Extraction automatique du numéro de dossier
  • Recherche du dossier client sur le Drive Carington
  • Enregistrement des pièces jointes dans le bon dossier
  • Mise à jour des tâches Asana (déplacement de section, suppression de champs personnalisés)
  • Suppression des mails traités (avec ou sans pièce jointe)

Utilisation

  1. Ouvrir l'application QuadraFlow
  2. Cliquez sur "Lancer le traitement"
  3. Consulter les logs en direct dans la console intégrée
