User Documentation - QuadraFlow

Overview

QuadraFlow is a desktop application designed to automatically process emails from a specific folder, extract attachments, sort them into the correct Drive folders, and update tasks in Asana.

Features

  • Reads emails from immatriculation@carington.fr
  • Automatically extracts the dossier number
  • Searches for the corresponding client folder in the Carington Drive
  • Saves attachments in the correct folder
  • Updates Asana tasks (section move, removal of custom field options)
  • Deletes processed emails (with or without attachment)

Usage

  1. Launch the QuadraFlow application
  2. Click "Start Processing"
  3. Monitor the live logs in the integrated console
