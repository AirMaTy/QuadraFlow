Documentation Technique - QuadraFlow

Architecture

  • Langage : Python 3.11
  • Librairies : imaplib, email, asana, tkinter, Pillow
  • Interface graphique : Tkinter
  • Build avec PyInstaller (inclusion de la documentation, icône personnalisée, logo)

Structure

  • QuadraFlow.py : Script principal
  • docs/ : Dossier contenant les fichiers HTML utilisateurs et techniques
  • QuadraFlow_logo.png : Logo utilisé dans l'application

Sécurité

Toutes les informations confidentielles sont stockées dans le fichier.env.
