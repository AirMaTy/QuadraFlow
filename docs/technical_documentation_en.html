Technical Documentation - QuadraFlow

Architecture

  • Language: Python 3.11
  • Libraries: imaplib, email, asana, tkinter, Pillow
  • GUI: Tkinter
  • Built with PyInstaller (includes documentation, custom icon, and logo)

Structure

  • QuadraFlow.py: Main script
  • docs/: Folder containing HTML user and technical documentation
  • QuadraFlow_logo.png: Logo used in the application

Security

All confidential information is stored in the.env file.
